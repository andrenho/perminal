Perminal - A multi-operation terminal
-------------------------------------

Prerequisites:
  - very fast startup (<10 ms)
  - plugin architecture
  - Lua interface
  - Multiprotocol (backend)
  - Multi-frontent
  - Client/server (?)
  - Settings simplified/complete

Features:
  - Use
    - Drag & drop
  - Session
    - Save/load sessions
    - Protocols: self-hosted unix, raw, telnet, ssh, scp, serial, DOS, etc... (as plugins)
    - Proxy
    - Logging
  - Window
    - Frontend: OpenGL, SDL, X11, Windows
    - Initial size (chars)
    - Initial position
    - Title
    - Scrollbar
      - Scrollback
    - Clock bar
    - Bell
    - Hide mouse when typing
    - Outer border
  - Console
    - Fonts
      - Antialiasing option
    - Cursor type
    - Colors
      - 256 colors
    - Transparency
    - Console plugin override (ex. time on screen)
    - Graphics on console
    - URL links
    - Key modifiers
    - Mouse
    - Clipboard area
  - Tabs
    - ???
  - Debugging
  - Advanced
    - Bidirectional text
    - Macros
    - Renderer filters
  - Documentation, manual page


ARCHITECTURE
------------
Computer <--> Plugin <--> Terminal <--> ServerCharMatrix <--> <<<<>>>>
                        (if printer)

                             <-- Input (Mouse/Keyboard)
<<<>>>> <-- ClientCharMatrix <--> Renderer
              |               --> Audio 
              `--> ResourceManager

Terminal types: matrix, printer
